# 校园食安卫士（Campus Food Safety Guardian）

> 面向校园食品安全治理、宣传教育与能力建设场景的前端展示型项目。

校园食安卫士是一个基于 **HTML + CSS + JavaScript** 实现的静态展示网站，围绕校园食品安全管理中的巡检督导、风险识别、宣传培训、整改闭环与数据支撑等核心需求，构建具有专业展示属性的整体方案页面。

该项目适用于：

- 课程设计与作业提交
- 比赛展示与答辩演示
- 项目申报与方案汇报
- 前端页面练习与原型展示
- GitHub / Gitee 项目作品集展示

## 项目简介

在校园食品安全治理实践中，常见问题包括：

- 巡检流程依赖人工记录，标准化与留痕能力不足
- 风险点识别不及时，预警机制相对薄弱
- 宣传教育和培训考核覆盖不充分
- 问题整改缺乏责任跟踪与闭环支撑
- 多角色协同和数据汇总效率不高

针对上述问题，本项目以“**发现问题—分析风险—组织培训—推动整改—持续优化**”为逻辑主线，设计并实现了校园食品安全能力建设整体方案展示页面，用于结构化表达项目内容与实施思路。

## 页面模块

本项目当前包含以下页面模块：

1. **首页展示（Hero Section）**
   - 项目名称
   - 核心定位
   - 核心价值概览
   - 行动按钮

2. **痛点分析**
   - 校园食品安全管理中的典型痛点归纳

3. **方案介绍**
   - 数字化巡检
   - 风险预警
   - 宣传教育
   - 培训考核
   - 整改闭环
   - 数据分析

4. **技术架构**
   - 感知采集层
   - 数据治理层
   - 业务支撑层
   - 应用展示层

5. **应用场景**
   - 学校食堂日常巡检
   - 从业人员培训考核
   - 校园食安宣传教育
   - 风险问题闭环管理
   - 教育系统专项督导
   - 多校区协同管理

6. **服务内容**
   - 需求调研
   - 方案设计
   - 培训与宣教内容建设
   - 部署与试运行
   - 评估优化与持续支持

7. **联系我们**
   - 联系信息展示
   - 咨询表单
   - 表单校验功能

## 快速开始

### 方法一：直接打开

直接双击项目中的 `index.html` 文件，即可在浏览器中预览页面。

### 方法二：使用 VS Code Live Server

推荐使用 VS Code 的 **Live Server** 插件：

1. 使用 VS Code 打开项目目录
2. 打开 `index.html`
3. 点击右下角 **Go Live**
4. 浏览器自动打开本地预览页面
